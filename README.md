# npx-template

deng-npx 模板

## 使用方法

#### 修改项目

文件   | 普通项目 | vue@1.x      | vue@2.x
------|---------|--------------|-----------
package.json | 清空依赖 | 使用 package.vue1.json 覆盖 package.json | 删除 package.vue1.json
.postcssrc.js| 删除 | 保留 | 删除

#### 安装依赖
```bash
npm install
```

#### 开始开发调试
```bash
npm run dev
```
